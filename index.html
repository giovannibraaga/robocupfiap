O Pitchulinho é um robô desenvolvido pela empresa "Eco-Sharpness". Ele está sendo desenvolvido para o projeto "RoboCup" da FIAP, O intuito do projeto é de que os alunos que cursam Sistemas de Informacoes trabalhem os conhecimentos adquiridos em todas as matérias do curso, para que no final possamos desenvolver um robô que seja capaz de participar de uma batalha para estourar o balão do oponente.
